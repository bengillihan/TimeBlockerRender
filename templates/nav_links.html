  • Cross-Application Embedding:
    1. Generate an embed token from the source application
    2. Copy the generated token and add it to your iframe URL: ?token=your_token_here
    3. The token will remain valid for 180 days
    4. Each application needs its own token
  • Internal Pages:
    1. For internal pages like Tasks or Calendar, use paths like '/tasks' or '/calendar'
    2. Click "Generate Embed Code" to get a pre-authenticated iframe
  • External Content:
    1. For external websites, use their provided embed code
    2. Paste it in the "Custom Iframe Code" field
    3. Note: External sign-in may not work in iframes due to security restrictions
